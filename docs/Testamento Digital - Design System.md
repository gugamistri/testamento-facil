4dp - Tight spacing (related form elements, chip gaps)
8dp - Small spacing (internal card padding elements)

32dp - Large spacing (major section separation)
48dp - Extra large spacing (screen padding top/bottom, hero sections)

Motion & Animation
Timing Functions
Standard Transition: 240ms, Ease-out cubic-bezier(0.4, 0.0, 0.2, 1)
 General UI state changes, hover effects


Deceleration: 280ms, Decelerate cubic-bezier(0.0, 0.0, 0.2, 1)
 Elements entering screen, modals opening


Acceleration: 200ms, Accelerate cubic-bezier(0.4, 0.0, 1, 1)
 Elements leaving screen, dismissing panels


Emphasis: 350ms, Spring cubic-bezier(0.34, 1.56, 0.64, 1)
 Success confirmations, premium features unlocked


Microinteractions: 180ms, Ease-in-out
 Button presses, checkbox toggles, switches


Specific Animations

Duration: 380ms

Behavior: Fade + 24dp Y-axis slide for new page entering

Success States

Additional Design Principles
Security Visual Language
Encryption indicators: Shield icons with Accent Emerald
Blockchain verification: Chain link icon with subtle gold accent
Multi-factor protection: Layered shield illustration
Status badges: Pill-shaped, 6dp corner radius, semi-bold text
Trust Signals
Legal validation markers: Checkmark + "ICP-Brasil Certified" badge
Timestamp displays: Monospace font for authenticity
Document versioning: Clear version numbers with date metadata
Security levels: Color-coded (Emerald = encrypted, Gold = blockchain verified)
Accessibility Requirements
Minimum contrast ratio: 4.5:1 for normal text, 3:1 for large text (WCAG AA)
Touch targets: Minimum 44dp × 44dp for all interactive elements
Focus indicators: 3dp Primary Deep Blue outline, 2dp offset
Screen reader: All icons include descriptive labels
Keyboard navigation: Logical tab order, visible focus states
